 GitGutter

Instalace
=========

::

   Plug 'airblade/vim-gitgutter'

Ovládání
========

* ``[c``

  * skoč na předchozí změnu v souboru

* ``]c``

  * skoč na další změnu v souboru

Konfigurace
===========

::

   " zrychli zobrazení diff značek

   set updatetime=250
